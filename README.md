# Basic Cafe Management System

This is a basic Cafe Management System developed using **Java, JSP, Servlets, and Hibernate**.  
The project provides simple user authentication and product management functionality.

---

## Technologies Used
- Java
- JSP
- Servlets
- Hibernate
- MySQL
- Apache Tomcat
- HTML / CSS
- Eclipse IDE

---

## Features
- Home Page
- User Login
- User Signup
- Contact Page
- Add Product
- Update Product
- Delete Product
